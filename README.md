# Deep Learning-Based Detection of Tuta absoluta
This repository contains the code and resources used for the development, evaluation, and deployment of deep learning models for the detection of *Tuta absoluta* under field conditions.
## Project Overview
The study focuses on automated identification of:
- Adult Tuta absoluta moths
- Tuta absoluta leaf damage symptoms
- Non-target instances
using lightweight convolutional neural network (CNN) architectures.
## Models Evaluated
- MobileNetV2
- EfficientNetV2s
- ResNet50
- ShuffleNetV2
- SqueezeNetV1.1
## Performance Evaluation
Models were evaluated using:
- Precision
- Recall
- F1-score
- Accuracy
- ROC-AUC
- Confusion Matrix
- Cochran's Q Test
- McNemar Test
## Deployment
The best-performing model was integrated into a web-based application for field-level pest detection and decision support.
